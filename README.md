## 🎨 Design Features

- **Professional-minimal** aesthetic (Latent Patterns style)
- **Clean typography** with system fonts only
- **White background** with black text and generous whitespace
- **High contrast** accessibility (WCAG AAA compliant)
- **No animations** or visual flourishes
- **Academic/professional** tone

## 🎯 Navigation

- **Desktop**: Arrow keys, spacebar, Home/End
- **Mobile**: Touch-friendly 44x44px buttons, swipe gestures
- **Keyboard accessible** with proper focus management
- **Slide counter** with current position

- **Mobile-first responsive design**
